# Roshan's Room

Right now this project is about **deciding on the furniture for Roshan's room**.

## Floorplan
- `index.html` — **interactive** to-scale 3D room (Three.js r184, orbit/zoom, dollhouse auto-hide walls). Primary view.
- `roshans_room_isometric.png` — static isometric cutaway export. Source: `tmp/draw_room.py` (`uv run tmp/draw_room.py`).
- Interior: **302 cm (length) × 215 cm (width) × 225 cm (height)**.
- Layout: bed on left wall (headboard against back wall/radiator, under the wall unit); door on front wall (front-left, opens in); main window + radiator on back wall; side window on right wall.
